# تطبيق المنتجات (Products App)

تطبيق أندرويد مبني بلغة Kotlin يتيح للمستخدم عرض المنتجات من الإنترنت بطريقة بسيطة ومرتبة،
مع دعم العمل بدون اتصال بالإنترنت من خلال التخزين المحلي باستخدام Room.

## الميزات
- جلب المنتجات من REST API باستخدام Retrofit
- عرض قائمة المنتجات باستخدام RecyclerView
- صفحة تفاصيل المنتج
- دعم الوضع Offline (عرض البيانات بدون إنترنت)
- استخدام معمارية MVVM
- التنقل بين الشاشات باستخدام Navigation Component
- Toolbar مع زر تحديث البيانات
- تحميل الصور باستخدام Glide

## التقنيات المستخدمة
- Kotlin
- Retrofit
- Room Database
- Coroutines
- MVVM Architecture
- Navigation Component
- Glide

## طريقة الاستخدام
1. عند فتح التطبيق يتم جلب المنتجات تلقائيًا من الإنترنت
2. يمكن الضغط على أي منتج لعرض التفاصيل
3. في حال عدم توفر الإنترنت يتم عرض البيانات المخزنة محليًا
4. يمكن استخدام زر التحديث لإعادة تحميل البيانات
